Etiquetas Contenedor

Las Etiqueras div y span de HTML sin elementos no semánticos, lo que quiere decir que no aportan ningun significado especifico sobre su contenido.

Se utilizan principalmente como contenedores generales para agrupar estilos con CSS o Funcionalidades con JavaScript.
